# Public NuShell Script 
## security
### GenCommand.nu script that generates a NuShell command.

This command is built from reading different type of security logs :
- suricata eve.json
- certipy output
     
This script:
- generates quoted columns
- plugs data holes
- flatten cells that contains [].
